WikiTree Formatting Reference

This reference helper is built using guidance directly from WikiTree's official style guides and media projects.

Section Headings

Headings must be bounded by double equal signs. Avoid single equal signs as they mess with the page title structure.

== Biography ==
== Sources ==
=== Timeline ===

Font Enhancements

''Italics''            -> ''text'' (2 apostrophes)
'''Bold'''             -> '''text''' (3 apostrophes)
'''''Bold & Italics''''' -> '''''text''''' (5 apostrophes)
<s>Strikeout</s>        -> <s>text</s> HTML tags

Citations & Footnotes

Always place citations inside the <ref> tag near the statement of fact. Ensure the <references/> tag sits underneath the Sources header.

He died in Poowong. <ref>Obituary, West Gippsland Gazette, 1901.</ref>

== Sources ==
<references/>
